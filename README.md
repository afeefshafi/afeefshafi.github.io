# Afeef Shafi - Personal Portfolio 🚀

A modern, fully responsive personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**.
This project showcases my skills, projects, and professional background with a clean UI, smooth animations, and real-world features like CV download and email contact.


## ✨ Features

- ⚡ Fast development & build using **Vite**
- 🎨 Modern UI with **Tailwind CSS**
- 📱 Fully responsive (mobile, tablet, desktop)
- 🧭 Advanced navbar with hamburger menu
- 🎬 Smooth animations using **Framer Motion**
- 📄 **CV Download** (PDF)
- 📧 **Working Contact Form** (EmailJS)
- 🔐 Secure environment variables
- 🌐 Deploymend on vercel


## 🛠 Tech Stack

- **Frontend:** React 18, Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **Animations:** Framer Motion
- **Email Service:** EmailJS
- **Icons:** React Icons
